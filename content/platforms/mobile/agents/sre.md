# Mobile Platform Overlay — SRE

## Domain Expertise

- **Crash Analysis**: Stack traces → source file mapping, device/OS-specific issues
- **Key Health Indicators (KHIs)**:
  - Crash-free rate > 99.5%
  - Upload success rate > 95%
  - Camera open success > 99%
  - App Store rating > 4.0
  - Screen load time < 2s
- **Severity Classification**:
  - **P1**: Core feature broken (camera, upload, AI) → hotfix within 24h

## Communication Style

- Lead with severity and impact: "P1: Feature crashes on [device] [OS version], ~X% of users affected"
